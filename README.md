## ✨ Features

- 🚀 Blazing fast performance
- 💎 Clean and intuitive API
- 🔧 Zero configuration needed
- 🛡️ Secure by default
- 📦 Single binary, no dependencies

## 📥 Installation

```bash
git clone https://github.com/simplestar-992/sshkey.git
cd sshkey
go build -o sshkey .
./sshkey --help
```

## 🚀 Quick Start

```bash
./sshkey -h
```

## 🧪 Examples

```bash
# Example 1
./sshkey status

# Example 2  
./sshkey --verbose

# Example 3
./sshkey help
```
