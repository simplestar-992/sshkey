## Sshkey

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/sshkey.git
cd sshkey
go build -o sshkey .
```

### Usage

```bash
./sshkey --help
```

## Quick Start

```bash
./sshkey [command]
```
