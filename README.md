# SSHKEY

**SSH Key Generator & Manager**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/sshkey.git
cd sshkey
go build -o sshkey .
```

## Usage

```bash
./sshkey --help
```
